# 🍎 Apple Global Sales Analysis

An end-to-end data analysis project focusing on Apple's global sales performance, customer behavior, and market trends.

## 🛠️ Tools Used
- **Python (Pandas):** For data cleaning and preprocessing.
- **SQL:** For performing business-driven data analysis.
- **Power BI:** For creating an interactive dashboard and data storytelling.

## 🚀 Project Workflow
1. **Data Cleaning:** Handled missing values and prepared the dataset using Pandas.
2. **SQL Analysis:** Extracted key metrics like total revenue, top countries, and product performance.
3. **Visualization:** Developed a Power BI dashboard to visualize KPIs and customer insights.

## 📊 Key Insights
- **Total Revenue:** $18.04M
- **Units Sold:** 23K
- **Customer Loyalty:** Significant migration from Android to iOS.
- **Average Rating:** 4.0/5.0

## 📂 Repository Content
- `apple_global_sales.csv` (Cleaned Data)
- `analysis_script.py` (Pandas Code)
- `queries.sql` (SQL Queries)
- `Apple_Dashboard.pbix` (Power BI File)
